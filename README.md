# quite
재고관리 프로젝트

기능 
1. 로그인 페이지
    1) 로그인, 로그아웃, 회원가입 (웹 연동), 아이디 찾기(웹 연동), 비밀번호 분실(웹 연동)
2. 초기 설정
    1) 마켓정보 등록 -> 셀렉트박스 , 마켓 관리자 계정 등록 -> 텍스트박스
3. 대시보드 -> 재고관리 탭 
    3-1. 등록
        1) 개별 등록(작성)
        2) 일괄 등록(업로드)
    3-2. 재고 현황
        1. 재고 수정, 삭제, 저장, 검색
4. 대시보드 -> 주문통합 탭
    4-1. 주문서
        1) 마켓 드롭 다운
        2) 주문 리스트 -> 물품코드, 주문자 정보, 송장정보 포함
            2-1) 주문 리스트 개별 수정, 삭제, 추가
    4-2. 분석
        1) 마켓별 판매 추이

페이지 
1. 로그인 페이지
2. 초기설정(마켓 정보) - 팝업
3. 메인화면 - 대시보드
    3-1. 재고관리
    3-2. 주문통합
        1) 주문서
        2) 분석
